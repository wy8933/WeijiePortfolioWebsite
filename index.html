About Me

Projects

No Right, Only Repair

No Right, Only Repair is a game made during the GMTK Game Jam 2024, I collaborated with a team of five to develop a game MVP under four days, following agile methodology and utilizing Trello for task management.

My primary contributions involved designing and implementing the user interface (UI) and scene management, which included creating the GUI, integrating player perks, building power upgrade interfaces, and developing a shop page using Unity's UI system and Scriptable Objects.

I also managed game events such as enemy waves and dynamic rule changes by using C# event handlers and delegates, ensuring smooth gameplay and responsive mechanics throughout the game.

Redemption Among Ruins

In Redemption Among Ruins, I played a key role in developing immersive game level environments and gameplay systems using Unreal Blueprints and C++. I focused on building responsive mechanics that enhanced player interaction and engagement.

Additionally, I created a comprehensive Game Design Document (GDD), detailing gameplay mechanics, level progression, and narrative elements to guide the development process. To further improve the player experience, I designed and implemented a user-friendly UI/UX, including intuitive game settings and GUI elements, all built using Unreal Blueprint.

Sword Rush

In Sword Rush, I played a key role in both the design and development of the game, contributing to gameplay mechanics, systems integration, and user experience. My responsibilities included:

  • Gameplay Programming: Developed core mechanics such as player movement, attack combos, and sword-based abilities, ensuring responsive controls for smooth gameplay.
  • Combat System: Implemented enemy AI behavior and designed engaging combat encounters, balancing difficulty to maintain player engagement.
  • Level Design: Collaborated on creating intricate levels and environments, integrating obstacles and rewards that align with the game's pacing and narrative.
  • UI/UX Implementation: Designed and developed in-game menus, health bars, and ability indicators to ensure clear communication between the player and game systems.
  • Collaboration: Worked closely with the team to align gameplay with the visual and narrative direction, using agile methodology for task management with tools like Trello and Git.
